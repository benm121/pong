# Pong

## Controls:
|Key    |Action |
| ----- | ----- |
|W      |Player 1 Up     |
|S      |Player 1 Down   |
|Up Arrow      |Player 2 Up    |
|Down Arrow      |Player 2 Down   |
|ESC    |Exit to menu    |

## Build and run:
git clone https://github.com/benm121/pong.git
cd pong && ./build.sh
./build/pong
